### description
<br />
Simple digital watchface, includes:
 - date and time
 - battery percentage
 - current location
 - current temperature
 - steps/sleep data
 - bluetooth symbol<br />
<br />
 
Options:
 - 24 or 12 hours
 - date order
 - location or health
 - fahrenheit or celsius
